### 核心功能支持

1. **用户系统**：
   - 用户注册/登录(users表)
   - 学习进度跟踪(user_progress表)
   - 个人学习统计
2. **背词模块**：
   - 单词库(vocabulary表)
   - 记忆算法实现(user_vocab_progress表记录熟悉度和复习时间)
   - 单词分类和难度分级
3. **答题模块**：
   - 题库系统(quiz_questions表)
   - 题目选项和解析(question_options表)
   - 答题记录(user_quiz_results表)
4. **阅读模块**：
   - 阅读材料库(reading_materials表)
   - 难度分级和分类
5. **听力模块**：
   - 听力材料库(listening_materials表)
   - 原文对照和难度分级

## 功能说明

这个日语学习网站包含以下核心功能：

1. **答题模块**：显示当前进度和剩余时间选择题形式，可标记答案上一题/下一题导航
2. **背词模块**：单词卡片展示（日语、读音、中文意思）例句和记忆技巧认识/不认识标记功能
3. **阅读模块**：日语文章阅读阅读理解题目选项选择功能
4. **听力模块**：音频播放区域（示例中为占位）听力原文显示听力理解题目
5. **通用功能**：侧边栏导航，显示用户信息和学习进度响应式设计，适配不同屏幕尺寸模块切换功能
